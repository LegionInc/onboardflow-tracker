This is a example for Page View and custom events

When you access this page, the event "Page View" will be registered. If you click on the link or the button the "Custom" event will be registered. 


                
                    <button onclick="custom_event('event_one', {'title': 'My Event One'})">
                        Custom Event One
                    </button>
                
            
                
                    <button onclick="custom_event('event_two', {'title': 'My Event Two'})">
                        Custom Event Two
